GTPProtocolのパイプライブラリです。

## How to use

1. DLLファイルをReleaseからダウンロードします。

   このとき、署名されたものを検証すると、ファイルが壊れていないかを確かめられます。

2. DLLファイルを参照します。

namespaceはGTP、クラス名はGTP2です。
